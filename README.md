# Luxe Bites

This Is an online website of food selling store that sells all types of foods in locals

- [Next.js](https://nextjs.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [Stripe](https://stripe.com)

## Features

- [x] Authentication
- [x] Authorization
- [x] Shopping Cart
- [x] Checkout
- [x] Orders
- [x] Payments
- [x] Admin Dashboard
- [x] User Dashboard
- [x] Responsive Design
- [x] SEO Friendly
